# Kit Base Moble v1

Este kit organiza o conhecimento oficial para o agente da Marcenaria Moble.

## Objetivo

Dar contexto consistente para o agente:

- entender o que a Moble faz;
- fazer triagem comercial curta e objetiva;
- qualificar leads;
- encaminhar para gerente de vendas quando necessario.

## Arquivos

- `base-oficial-moble.md`: base consolidada oficial da Moble (versao principal).
- `base-negocio.md`: posicionamento, escopo de servicos e politicas.
- `playbook-atendimento.md`: fluxo operacional de atendimento.
- `faq-comercial.md`: respostas curtas para perguntas frequentes.
- `roteiro-qualificacao.md`: perguntas padrao para primeiro atendimento.

## Como usar

1. Revisar e ajustar os dados para a realidade da Moble.
2. Subir os arquivos no OneDrive (pasta dedicada, ex.: `Moble/Atendimento/`).
3. Manter estes arquivos atualizados sempre que houver mudanca de processo, prazo ou politica.
